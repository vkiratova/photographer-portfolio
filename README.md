\*\*Вариант 1. Frontend-разработка\*\*  

Выполнено за 5 дней согласно заданию.



\## Что сделано

\- Семантическая HTML-разметка (header, main, section, footer и т.д.)

\- Адаптивная вёрстка на Flexbox + медиа-запросы (десктоп → планшет → мобильные)

\- Подключены Google Fonts (Roboto)

\- 4 интерактивных элемента на чистом JavaScript:

&nbsp; 1. Бургер-меню для мобильной версии

&nbsp; 2. Табы в разделе «Услуги»

&nbsp; 3. Модальное окно с формой обратной связи

&nbsp; 4. Слайдер галереи (4 фотографии)

\- Оптимизация: код прошёл валидацию W3C, изображения по внешним ссылкам (в продакшене можно скачать локально)

\- Никаких фреймворков и библиотек — чистый HTML/CSS/JS



\## Как запустить

Просто открыть файл `index.html` в любом браузере.
